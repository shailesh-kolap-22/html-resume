# HTML Resume

A professional resume built using HTML to demonstrate fundamental HTML knowledge.

## About

This is a simple, HTML resume showcasing:
- Proper HTML5 structure
- Semantic elements (header, main, footer, section, article)
- Tables with thead and tbody
- Links and contact information

## Features

- Semantic HTML5 elements
- Structured tables for experience and education
- External links to projects and social profiles
- Clean and organized code with comments

## Technologies

- HTML5

## How to View

Open `index.html` in any web browser.
